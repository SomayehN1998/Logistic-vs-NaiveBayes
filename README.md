# 📊🤖 **Veros-ML-Models-Comparison**: مقایسه مدل‌های ماشین لرنینگ

این پروژه یک **مقایسه حرفه‌ای** بین دو مدل محبوب ماشین لرنینگ—**رگرسیون لجستیکی (Logistic Regression)** و **شبکه‌های بیز (Naive Bayes)**—برای پیش‌بینی ستون **OUTPUT** در دیتاست **anti-malware.csv** است. هدف پروژه طبقه‌بندی خروجی (classifying the target) است.

## 📝 **توضیح پروژه**

این پروژه به منظور مقایسه عملکرد دو الگوریتم طبقه‌بندی طراحی شده و شامل مراحل زیر است:

### بخش‌های پروژه:
- **بخش الف**: بارگذاری داده‌ها، پیش‌پردازش (مدیریت مقادیر گمشده، کدگذاری ویژگی‌ها) و تقسیم داده‌ها.
- **بخش ب**: آموزش و ارزیابی **رگرسیون لجستیکی** با استفاده از **ماتریس درهم‌آمیزی**.
- **بخش ج**: آموزش و ارزیابی **Naive Bayes (GaussianNB)** با استفاده از **ماتریس درهم‌آمیزی**.
- **بخش د**: مقایسه عملکرد دو مدل بر اساس **دقت (Accuracy)**.
